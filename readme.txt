== Description ==

Demo: http://clownfishweb.com/demo/responsive-carousel-slider/

= Responsive Carousel Slider =

- 100% Responsive Slider extension.
- Highly customizable design.
- Looks awesome on any device.
- Ability to change Pagination Speed.
- Ability to change slider transition duration.
- Open image link in new tab or same tab.
- Can select auto slider, display controls, loop and many more.

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/PLUGIN_NAME' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
